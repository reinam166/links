# Number row characters 
- `!`		-Exclmation mark/point
- `@`		-At sign
- `#`		-Hashtag, pound key
- `$`		-Dollar Sign
- `%`		-Percent
- `^`		-Caret
- `&`		-Ampersand, And Sign
- `*`		-Asterisk
- `(`		-Open Bracket
- `)`		-Close Bracket
